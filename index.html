HTML Table Cheatsheet

Table Tags

Tag Name Description
<table> Table Елемент обгортки для всіх HTML-таблиць.
<thead> Table Head Набір рядків, що визначають заголовки стовпців у таблиці.
<tbody> Table Body Набір рядків, що містять фактичні дані таблиці.
<tr> Table Row Контейнер рядка таблиці.
<td> Table Data Контейнер у рядку для даних.
<th> Table Header Призначений для створення однієї комірки-заголовка таблиці.
<tfoot> Table Foot Набір рядків, що визначають нижній колонтитул у таблиці.

Table Attributes

Attribute Name Description
colspan Column Span Визначає, скільки стовпців повинен займати елемент td.
rowspan Row Span Визначає, скільки рядків повинен займати елемент td.
